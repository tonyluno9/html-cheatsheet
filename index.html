Cheat Sheet de HTML

Tip de examen: recuerda que las etiquetas semánticas (<header>, <main>, etc.) ayudan a accesibilidad y SEO.
1) Documento y estructura básica
<!DOCTYPE html>
Indica al navegador que el documento sigue el estándar HTML5.
<!DOCTYPE html>
<html>
Elemento raíz que envuelve todo el documento (excepto el doctype).
<html lang="es">
  
</html>
<head>
Contiene metadatos: codificación, título, enlaces a CSS/JS, meta etiquetas.
<head>
  <meta charset="UTF-8">
  <title>Mi página</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
Incluye todo el contenido visible (texto, imágenes, secciones, formularios, etc.).
<body>
  <h1>Hola, mundo</h1>
</body>
<title>
Define el título que aparece en la pestaña del navegador o al marcar favoritos.
<head>
  <title>Portafolio de Ana</title>
</head>
2) Secciones semánticas
<header>
Encabezado de una página o sección. Suele contener logotipo, título y/o navegación principal.
<header>
  <h1>Mi Blog</h1>
  <p>Historias y tutoriales</p>
</header>
<nav>
Bloque de enlaces de navegación (principal o secundaria).
<nav aria-label="Principal">
  <a href="/">Inicio</a>
  <a href="/articulos">Artículos</a>
  <a href="/contacto">Contacto</a>
</nav>
<main>
Contenido principal único de la página (debe haber solo un <main> por documento).
<main>
  <article></article>
</main>
<aside>
Contenido complementario o lateral (widgets, anuncios, enlaces relacionados).
<aside>
  <h2>Populares</h2>
  <ul><li>Post A</li></ul>
</aside>
<footer>
Pie de página. Suele incluir derechos, enlaces legales o información de contacto.
<footer>
  © 2025 Mi Sitio — <a href="/privacidad">Privacidad</a>
</footer>
3) Texto, contenedores y enlaces
<h1> – <h6>
Encabezados jerárquicos del más importante (<h1>) al menos (<h6>). Útiles para SEO y estructura.
<h1>Título de la página</h1>
<h2>Sección</h2>
<h3>Subsección</h3>
<p>
Párrafo de texto.
<p>Este es un párrafo con una idea completa.</p>
<span>
Contenedor en línea para aplicar estilos o agrupar texto sin romper el flujo.
<p>Precio: <span style="color:#0bd67a">$199</span></p>
<div>
Contenedor en bloque genérico para agrupar elementos.
<div class="tarjeta">
  <h3>Curso HTML</h3>
  <p>Aprende desde cero.</p>
</div>
<a>
Enlace a otra página, sección o recurso. Usa href; target="_blank" abre en nueva pestaña.
<a href="https://developer.mozilla.org/" target="_blank" rel="noopener">
  Documentación MDN
</a>
4) Listas y definiciones
<ol>
Lista ordenada (numerada).
<ol>
  <li>Instalar</li>
  <li>Configurar</li>
  <li>Ejecutar</li>
</ol>
<ul>
Lista desordenada (viñetas).
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JS</li>
</ul>
<dl>
Lista de definiciones (término y descripción). (Etiqueta correcta en vez de <di>)
<dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado para la web.</dd>
</dl>
<li>
Elemento de lista dentro de <ol> o <ul>.
<ul>
  <li>Elemento A</li>
  <li>Elemento B</li>
</ul>
<dt>
Término dentro de una lista de definiciones (<dl>).
<dl>
  <dt>API</dt>
  <dd>Interfaz para que programas se comuniquen.</dd>
</dl>
<dd>
Descripción del término (<dt>) dentro de <dl>.
<dl>
  <dt>CSS</dt>
  <dd>Hojas de estilo en cascada.</dd>
</dl>
5) Tablas
<table>
Tabla que organiza datos en filas y columnas.
<table></table>
Usa <caption> para un título opcional de la tabla.
<thead>
Sección de encabezado de la tabla. Suele contener los títulos de columna.
<thead>
  <tr><th>Nombre</th><th>Edad</th></tr>
</thead>
<tbody>
Cuerpo principal de datos de la tabla.
<tbody>
  <tr><td>Ana</td><td>20</td></tr>
</tbody>
<tfoot>
Pie de la tabla (sumas, totales u observaciones).
<tfoot>
  <tr><td colspan="2">Total: 1 registro</td></tr>
</tfoot>
<tr>
Table Row: fila de la tabla.
<tr>
  <td>Dato1</td><td>Dato2</td>
</tr>
<th>
Table Header: celda de encabezado (por defecto en negrita/centrado).
<tr>
  <th scope="col">Producto</th>
  <th scope="col">Precio</th>
</tr>
<td>
Table Data: celda de datos.
<tr>
  <td>Teclado</td><td>$500</td>
</tr>
6) Multimedia
<img>
Inserta una imagen. alt describe la imagen para accesibilidad.
<img src="foto.jpg" alt="Persona sonriendo" width="320" height="180">
<audio>
Inserta audio. Con controls muestra los controles del reproductor.
<audio controls>
  <source src="tema.mp3" type="audio/mpeg">
  Tu navegador no soporta audio.
</audio>
<video>
Inserta video. Acepta atributos como controls, autoplay, loop.
<video width="400" controls>
  <source src="intro.mp4" type="video/mp4">
  Tu navegador no soporta video.
</video>
<source>
Define una fuente de medios alternativa dentro de <audio> o <video> (diferentes formatos).
<video controls>
  <source src="clip.webm" type="video/webm">
  <source src="clip.mp4"  type="video/mp4">
</video>
7) Formularios
<form>
Agrupa controles de entrada. Usa action (destino) y method (GET/POST).
<form action="/buscar" method="get">
</form>
<label>
Etiqueta para un control de formulario. Asocia con for = id del control.
<label for="email">Correo:</label>
<input id="email" type="email">
<input>
Campo de entrada. Tipos comunes: text, email, number, password, checkbox, radio, date, etc.
<input type="text" name="nombre" placeholder="Tu nombre" required>
<input type="checkbox" name="acepto"> Acepto
<input type="radio" name="plan" value="pro"> Pro
<select> & <option>
Lista desplegable de opciones.
<label for="pais">País:</label>
<select id="pais" name="pais">
  <option value="mx">México</option>
  <option value="ar">Argentina</option>
</select>
<textarea>
Área de texto multilínea.
<textarea name="mensaje" rows="4" cols="30" placeholder="Escribe tu mensaje..."></textarea>
